Mau bikin server Minecraft kamu makin keren? Gampang banget, kok! Simak panduan instal plugin berikut, dijamin simpel dan no ribet.

## Sebelum Mulai, Pilih Versi Server yang Tepat

Pastikan kamu pakai versi server Minecraft yang support plugin, ya. Paling aman dan recommended banget: **[Paper](https://papermc.io/)** atau **[Purpur](https://purpurmc.org/download/purpur/)**. 

Kenapa harus Paper/Purpur? Karena:
- Stabil banget
- Selalu update
- Support plugin dari Bukkit, Spigot, dll.

Kalau kamu masih pakai **Bukkit atau Spigot**, coba deh upgrade ke Paper/Purpur. 

> ⚠️ **Note penting:** Server mod kayak Forge atau Fabric **nggak bisa** jalanin plugin, ya. Hybrid server yang klaim bisa dua-duanya biasanya nggak stabil. Jadi mending pilih salah satu.

---

## Cara Instal Plugin Minecraft (Step-by-Step)

Udah siap? Yuk, langsung eksekusi! 💻

### 1. Cari Plugin yang Aman 🔍
Ada banyak situs buat download plugin. Ini beberapa yang recommended:
- [SpigotMC](https://www.spigotmc.org)
- [Bukkit](https://dev.bukkit.org)
- [Modrinth](https://modrinth.com)

Hindarin situs abal-abal yang nawarin plugin bajakan. Selain nggak etis, risiko kena virus tuh gede banget. 😬

<img src="images/spigotmc.png" alt="SpigotMC Website" style={{ width: '100%', height: 'auto' }} />

### 2. Cek Kompatibilitas Plugin ✅
Sebelum download, cek dulu apakah plugin itu support versi Minecraft server kamu. Biasanya ada info di bagian **"Tested Minecraft Versions"**. Kalau versi utama (misalnya 1.20) didukung, biasanya versi kecilnya (kayak 1.20.2) juga aman.

### 3. Download Plugin 💾
Klik tombol **"Download Now"** dan simpan file .jar plugin ke komputer kamu.

<img src="images/files-panel.png" alt="File Manager Location" style={{ width: '100%', height: 'auto' }} />

### 4. Upload Plugin ke Server 📤

1. Pastikan server kamu dalam kondisi mati (shutdown). Ini penting biar semua data aman.
2. Buka **File Manager** di panel server kamu.
3. Cari folder **"plugins"**. Kalau nggak ada, coba jalankan server dulu sekali biar foldernya muncul.

<img src="images/files.png" alt="Plugins Folder" style={{ width: '100%', height: 'auto' }} />

4. Drag and drop file .jar plugin ke dalam folder "plugins" di panel server.

<img src="images/upload-plugin.gif" alt="Upload Plugin" style={{ width: '100%', height: 'auto' }} />

### 5. Restart Server 🔄

Setelah file di-upload, nyalain server kamu lagi. Kalau semua berjalan lancar, ketik **"pl"** di konsol server. Nanti plugin yang baru kamu instal bakal muncul di daftar.
