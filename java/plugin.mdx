## Before Starting, Choose the Right Server Version

Make sure you're using a Minecraft server version that supports plugins. The safest and most recommended options are: [Paper](https://papermc.io/) or [Purpur](https://purpurmc.org/download/purpur/).

Why choose Paper/Purpur? Because:

* Very stable

* Always updated

* Supports plugins from Bukkit, Spigot, etc.

If you're still using **Bukkit or Spigot**, try upgrading to Paper/Purpur.
