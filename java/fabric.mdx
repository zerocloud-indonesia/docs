Mau eksplorasi Minecraft dengan mod yang keren? Berikut panduan mudah buat instal mod menggunakan **Fabric** di server Minecraft kamu.

## Sebelum Mulai, Pastikan Persiapan Sudah Lengkap

1. **Gunakan versi Minecraft yang kompatibel dengan Fabric.**
2. Download dan instal **[Fabric Loader](https://fabricmc.net/use/)** di komputer kamu.

<img src="/images/fabricmc.png" alt="Fabric Installer" style={{ width: '100%', height: 'auto' }} />

3. Siapkan mod yang ingin digunakan dari sumber terpercaya seperti **[Modrinth](https://modrinth.com)**.

> ‚ö†Ô∏è **Catatan penting:** Server dengan Fabric **tidak mendukung plugin Bukkit atau Spigot**. Kalau ingin hybrid, kamu bisa gunakan mod tambahan seperti **Mohist** (dengan risiko kestabilan).

---

## Cara Instal Mod Fabric di Minecraft (Step-by-Step)

### 1. Instal Fabric Loader di Server

1. Buka [Fabric Installer](https://fabricmc.net/use/).
2. Pilih versi Minecraft yang sesuai dengan server kamu.
3. Klik **Install Server** dan pilih direktori tempat server Minecraft kamu berada.

<img src="/images/fabricinstall.png" alt="Fabric Installer" style={{ width: '100%', height: 'auto' }} />

4. Jalankan file **fabric-server-launch.jar** untuk membuat file dan folder default server Fabric.

---

### 2. Download Mod dari Modrinth

1. Buka [Modrinth](https://modrinth.com) untuk mencari mod yang kompatibel dengan Fabric.

<img src="https://cdn-raw.modrinth.com/app-landing/app-screenshot.webp" alt="Modrinth" style={{ width: '100%', height: 'auto' }} />

2. Gunakan filter untuk memilih mod versi Minecraft yang kamu gunakan.
3. Klik **Download** dan simpan file .jar mod di komputer kamu.

<img src="/images/modrinth-home.png" alt="Modrinth Website" style={{ width: '100%', height: 'auto' }} />

> **Tips:** Jangan lupa cek dependensi mod di bagian deskripsi Modrinth. Kadang mod membutuhkan library tambahan seperti **Fabric API**.

---

### 3. Upload Mod ke Server

1. Pastikan server dalam kondisi mati (shutdown).
2. Buka **File Manager** di panel server kamu.
3. Cari folder **mods**. Kalau belum ada, buat folder baru dengan nama "mods".
4. Upload semua file .jar mod (termasuk Fabric API jika diperlukan) ke dalam folder **mods**.

<img src="/images/mods-folder.png" alt="Folder Mods" style={{ width: '100%', height: 'auto' }} />

---

### 4. Jalankan Server

1. Restart server untuk mengaktifkan mod yang sudah di-upload.
2. Buka konsol server dan cek apakah semua mod sudah terdeteksi tanpa error.
3. Jika berhasil, modifikasi server sudah siap digunakan! üöÄ
