Mau bikin server impianmu di ZeroCloud? Ikuti langkah-langkah berikut untuk membeli saldo dan memesan hosting.

## Cara Membeli Saldo di ZeroCloud Indonesia

<Steps>
   <Step title="Registrasi atau Login ke Billing ZeroCloud">
      Registrasi atau login ke akun kamu di [billing.zerocloud.id](https://billing.zerocloud.id).
      ![Tampilan Login Billing](/images/login.gif)
   </Step>
   <Step title="Buka Menu Store">
      Buka menu **Store**, pilih jumlah saldo yang ingin diisi, lalu tekan tombol **Purchase**.
      ![Menu Store](/images/store.png)
   </Step>
   <Step title="Pilih Metode Pembayaran">
      Pilih metode pembayaran favorit kamu (Midtrans/QRIS) dan klik **Submit Payment**.
      ![Metode Pembayaran](/images/payment.png)
   </Step>
   <Step title="Lakukan Pembayaran">
      Akan muncul QRIS yang bisa langsung kamu scan. Lakukan pembayaran sesuai petunjuk.
      ![QRIS Pembayaran](/images/qris.gif)
      Jika pembayaran berhasil, saldo kamu otomatis bertambah.
   </Step>
</Steps>

---

## Cara Memesan Server di ZeroCloud

<Steps>
   <Step title="Buka Menu Create Server">
      Pergi ke menu **Create Server** di panel ZeroCloud.
      ![Menu Create Server](/images/create-server.png)
   </Step>
   <Step title="Isi Detail Server">
      Isi nama server dan parameter lainnya sesuai kebutuhanmu.
      ![Form Create Server](/images/server-form.png)
   </Step>
   <Step title="Pilih Software/Games">
      Pilih **Software/Games** dan set default ke **Onboarding** (bisa diubah nanti di panel).
      ![Pilih Software/Games](/images/software-selection.png)
   </Step>
   <Step title="Pilih Node">
      Pilih Node yang tersedia. Pastikan stok cukup sebelum melanjutkan.
      ![Pilih Node](/images/node-selection.png)
   </Step>
   <Step title="Konfirmasi untuk Private Node">
      Untuk Private Node, silakan konfirmasi melalui tiket.
   </Step>
   <Step title="Selesaikan Pemesanan">
      Server kamu akan siap digunakan. Setelah selesai, tekan tombol **Manage** atau login ke Control Panel untuk mulai eksplorasi server kamu.
   </Step>
</Steps>
